# Nuxt Isomorphic App

This project consist of two Node.js applications:
- backend
- frontend 

## Install dependencies for backend app:
``` cli
npm i back-install
```

## Install dependencies for frontend app:
``` cli
npm i front-install
```

## Start backend app
``` cli
npm run back-start
```

## Start frontend app
``` cli
npm run front-dev
```
or
``` cli
npm run front-build
npm run front-start
```
